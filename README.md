# Ubimate — Releases

Official desktop app downloads for **[Ubimate](https://ubimate.com)**, a privacy-oriented, local-first alternative to Notion.

This repository hosts the **published binaries** only — it exists purely to distribute release assets that anyone can download without a GitHub account.

## Download

Grab the latest build from the **[Releases page](https://github.com/ubimate/ubimate-release/releases/latest)**, or use the download picker on **[ubimate.com/download](https://ubimate.com/download)** which auto-detects your platform.

| Platform | File |
|----------|------|
| macOS (Apple Silicon) | `Ubimate_X.Y.Z_aarch64.dmg` |
| macOS (Intel) | `Ubimate_X.Y.Z_x64.dmg` |
| Windows (x64) | `Ubimate_X.Y.Z_x64_en-US.msi` |

> Not sure which Mac you have? Click the Apple menu → **About This Mac**. "Apple M1/M2/M3…" means Apple Silicon; "Intel" means the Intel build.

## Installation

### macOS

1. Download the `.dmg` for your chip.
2. Open it and drag **Ubimate** into your **Applications** folder.
3. Launch it from Applications.

If macOS warns that the app is from an unidentified developer, right-click the app → **Open**, then confirm.

### Windows

1. Download the `.msi`.
2. Run the installer and follow the prompts.

If SmartScreen appears, click **More info → Run anyway**.

## What is Ubimate?

Ubimate is a **local-first** notes and documents app:

- **Your data lives on your device.** No account or internet connection is required to use the app — everything is stored locally.
- **Optional cloud sync.** A paid subscription enables real-time collaboration and multi-device sync.
- **Rich editing.** Pages, notes, tables, media embeds, and more.

Learn more at **[ubimate.com](https://ubimate.com)**.
